# 摸鱼搭子微信小程序

这是一个原生微信小程序 MVP，围绕上班激励、摸鱼陪伴、下班找搭子和生活兴趣展开。

## 页面模块结构

页面按业务模块分组，每个模块下再按功能独立建文件夹：

```text
pages/
  today/
    home/          今日首页
  pet/
    home/          宠物小窝
  fish/
    home/          摸鱼广场
  work/
    salary/        工资计时和掉钱动画
  offwork/
    home/          下班首页
  profile/
    home/          我的页面
```

## 已包含功能

- 今日：首页展示今日已赚、今日进度、宠物陪伴和快捷入口
- 鱼塘：上班摸鱼动态，支持发动态、评论、点赞、收藏和同款
- 宠物：独立宠物小窝，承载互动、养成、装扮等能力
- 工资计时：点击今日进度进入掉钱页面，可设置工资和上下班时间
- 下班：约打球、娱乐麻将、今天吃什么、搭子广场
- 我的：工资设置展示、宠物状态、我的约局、成就徽章
- 后续扩展：玄学分享、AI 音乐创作分享等内容可以接入鱼塘动态

## 如何运行

1. 打开微信开发者工具
2. 选择“导入项目”
3. 项目目录选择当前文件夹
4. AppID 可以先使用测试号或填你自己的小程序 AppID
5. 点击编译预览

## 后续能力接入

AI 音乐创作、玄学分享等能力建议先作为鱼塘动态的内容类型接入。涉及 AI 平台密钥时，不要放在小程序前端代码里，应该通过云函数或自己的后端转发请求。
